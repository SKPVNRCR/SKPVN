We use GitHub Issues to collect submissions for the SKPVN Smriti magazine. Submitting via issues is free, secure and works seamlessly on GitHub Pages.

Open Submission Form View All Submissions

How publishing works: New submissions are created with the label pending. An admin reviews each submission and adds the label approved when it is ready for publication. Approved articles and poems appear automatically on the Articles page.

  • Submitters will need a free GitHub account.
  • No data is stored in your browser anymore—everything lives on GitHub.
